# panda-generator

#### 项目介绍
    本项目可以根据数据模型生成80%的代码。
    本项目在人人开源项目的基础上优化了读取模板的操作，作者在使用renren-generator时新编写的模板配置过于繁琐所以进行了改造；
    新增insert SQL脚本生成功能（方便导数据）
#### 软件架构
